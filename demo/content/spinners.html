Spinner

<au-spinner> is an MD3 circular progress indicator for short loading operations where the wait time is unknown.

OVERVIEW API STYLING EXAMPLES
Small Medium Large
<au-spinner size="sm"></au-spinner> <au-spinner size="md"></au-spinner> <au-spinner size="lg"></au-spinner>
<au-spinner color="primary"></au-spinner> <au-spinner color="secondary"></au-spinner> <au-spinner color="tertiary"></au-spinner>

When to use

Use spinners for short, indeterminate loading operations (under 4 seconds). For longer operations, use a progress indicator with percentage. For content areas, prefer skeletons over spinners.

Attributes

'sm' | 'md' | 'lg' 'primary' | 'secondary' | 'tertiary'
