Radio Button

<au-radio-group> and <au-radio> allow users to select one option from a set. Radio buttons should be used in groups.

OVERVIEW API STYLING EXAMPLES
Small Medium Large

Accessibility

Radio groups use role="radiogroup" and manage aria-checked on each radio. Arrow keys navigate between options.
