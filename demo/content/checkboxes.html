Checkbox

Checkbox label

The checkbox label is provided as the content to the <au-checkbox> element. Alternatively, use the label attribute. For accessibility, always provide a label.

Accessibility

<au-checkbox> uses a native role="checkbox" and manages aria-checked automatically based on state. The indeterminate state uses aria-checked="mixed".
