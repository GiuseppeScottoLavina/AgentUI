Skeleton

<au-skeleton> displays animated placeholder content while the actual content is loading, improving perceived performance.

When to use

Use skeletons to indicate that content is loading. They reduce perceived waiting time by showing the layout structure upfront. Prefer skeletons over spinners for content-heavy areas.

Attributes

'rect' | 'circle' | 'text' Width for rect variant Height for rect variant Size for circle variant Number of lines for text variant
