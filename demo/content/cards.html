Card

<au-card> is a surface container that holds related content. MD3 defines 3 card variants with different elevation and emphasis levels.

OVERVIEW API STYLING EXAMPLES

Elevated

With shadow for depth.

Filled

Surface container color.

Outlined

With border, no shadow.

Choosing a card variant

Elevated High emphasis - main content, hero sections Filled Medium emphasis - grouped items, list items Outlined Low emphasis - subtle containers, nested cards

Attributes

Card style: 'elevated' | 'filled' | 'outlined' Whether card is clickable with hover state
