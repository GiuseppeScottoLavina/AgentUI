Toast / Snackbar

showToast() displays temporary notification messages that auto-dismiss after a configurable duration.

OVERVIEW API STYLING EXAMPLES
Success Info Warning Error
showToast('Success!', { severity: 'success' }); showToast('Error occurred', { severity: 'error' });

When to use

Brief, non-critical feedback
"Saved", "Copied to clipboard", "Settings updated"
Action confirmation
User just clicked something, needs visual acknowledgment
With undo action
Reversible actions like "Item deleted" with an Undo button

When NOT to use

Don't use for errors requiring user action — use <au-alert> inline so user sees it while correcting Don't use for important status always visible — toasts disappear! Use <au-alert> instead Don't use for confirmations requiring decision — use <au-confirm> which blocks until user responds

Function Signature

showToast(message: string, options?: ToastOptions): void

Options

Toast type: 'success' | 'info' | 'warning' | 'error' Auto-dismiss duration in ms Optional action button text

CSS Custom Properties

Toast background Toast text color
Delete with Undo
