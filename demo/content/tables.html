Table

<au-table> provides semantic HTML table elements with MD3 styling for displaying tabular data.

OVERVIEW API STYLING EXAMPLES
Name Email Role John Doe john@example.com Admin Jane Smith jane@example.com Editor Bob Johnson bob@example.com Viewer
<au-table> <au-thead> <au-tr><au-th>Name</au-th><au-th>Email</au-th></au-tr> </au-thead> <au-tbody> <au-tr><au-td>John</au-td><au-td>john@example.com</au-td></au-tr> </au-tbody> </au-table>

Semantic structure

Use <au-thead> for headers, <au-tbody> for data, <au-tr> for rows, <au-th> for header cells, and <au-td> for data cells.

Elements

Table container (display: table) Table header group Table body group Table row Header cell (bold, left-aligned) Data cell
