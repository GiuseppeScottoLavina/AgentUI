Tooltip

<au-tooltip> displays contextual information when users hover or focus on an element.

OVERVIEW API STYLING EXAMPLES
Top Bottom Left Right
<au-tooltip content="Help text" position="top"> <au-button>Hover me</au-button> </au-tooltip>
<au-tooltip content="Edit document"> <au-icon name="edit"></au-icon> </au-tooltip>

When to use

Use tooltips to provide additional context for icon buttons, truncated text, or any element where a brief explanation improves usability. Keep tooltip text conciseâ€”one or two words is ideal.

Attributes

The tooltip text to display 'top' | 'bottom' | 'left' | 'right'
