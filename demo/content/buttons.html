Button

<au-button> is an interactive element that triggers an action. Material Design 3 defines 5 button types for different emphasis levels.

OVERVIEW API STYLING EXAMPLES
Filled Elevated Tonal Outlined Text
Small Medium Large

Choosing a button type

Filled High emphasis - primary actions (Save, Submit, Confirm) Tonal Medium emphasis - secondary actions (Add, Edit) Outlined Medium emphasis - alternative to tonal Text Low emphasis - tertiary actions (Cancel, Learn more)

Accessibility

<au-button> uses role="button" and manages tabindex automatically. Keyboard navigation with Enter/Space is built-in. The ripple effect provides visual feedback for all interactions.

Selector

<au-button>

Attributes

Button style: 'filled' | 'elevated' | 'tonal' | 'outlined' | 'text' | 'danger' Button size: 'sm' | 'md' | 'lg' Whether the button is disabled

Events

Native click event (works with disabled state)

CSS Classes

Base class for all buttons Filled variant styling Elevated variant with shadow Tonal (secondary container) styling Outlined variant styling Text-only variant styling Applied when button is disabled

CSS Custom Properties

Customize buttons using MD3 design tokens.

Filled button background Filled button text color Tonal button background Tonal button text color Outlined button border Button border radius Elevated button shadow

Theming Example
