Text Field

<au-input> and <au-textarea> allow users to enter and edit text. Both support MD3 variants with floating labels and focus animations.

OVERVIEW API STYLING EXAMPLES

Accessibility

Always provide a label attribute for screen readers. The floating label animates when focused, providing clear visual feedback. Focus states use box-shadow instead of border-width to prevent layout shifts.

<au-textarea>
