Switch

<au-switch> is a toggle control for binary on/off settings. It follows MD3 specifications with smooth thumb animations and state colors.

OVERVIEW API STYLING EXAMPLES

Accessibility

<au-switch> uses role="switch" and manages aria-checked automatically. Keyboard toggle with Space is supported.
