Confirm Dialog

<au-confirm> is a Promise-based confirmation dialog. Use auConfirm() for one-line confirm prompts.

OVERVIEW API STYLING EXAMPLES
Delete Item
import { auConfirm } from 'agentui-wc'; const ok = await auConfirm('Delete this item?', { title: 'Confirm Delete', confirmText: 'Delete', variant: 'danger' }); if (ok) deleteItem();

Promise-based API

auConfirm() returns a Promise that resolves to true when confirmed, or false when cancelled. Use with async/await for clean one-liner confirmations.

Function Signature

auConfirm(message: string, options?: ConfirmOptions): Promise<boolean>

Options

Dialog title Confirm button text Cancel button text 'primary' | 'danger' | 'warning'

Events

Emitted when user confirms Emitted when user cancels (or presses ESC)

CSS Classes

Backdrop overlay Dialog container Title element Message text Button container
