Callout

<au-callout> is an MD3 styled info box for highlighting important information, tips, warnings, or success messages in documentation.

OVERVIEW API STYLING EXAMPLES
Note: This is an informational callout for general guidance. Tip: Use keyboard shortcut Ctrl+S to save your work quickly. Warning: This action cannot be undone. Please proceed with caution. Success: Your changes have been saved successfully.
<au-callout variant="info">Info message</au-callout> <au-callout variant="tip">Tip message</au-callout> <au-callout variant="warning">Warning message</au-callout> <au-callout variant="success">Success message</au-callout>

When to use

Use callouts to draw attention to important information that should stand out from regular content. Unlike alerts which are for user feedback, callouts are ideal for documentation, tutorials, and static informational content.

Attributes

'info' | 'tip' | 'warning' | 'success'

CSS Custom Properties

Info background Tip background Warning background Success background

When implementing authentication, consider the following:

Always use HTTPS in production to protect user credentials during transmission.

For token storage, we recommend using HttpOnly cookies:

HttpOnly cookies prevent XSS attacks from stealing authentication tokens via JavaScript.
